# Todo-List

Приложение **TodoList** — простое приложение для ведения списка дел(ToDo List) с возможностью добавления, редактирования, поиска, удаления задач.

# Используемый API

DummyJSON - Free Fake REST API for Placeholder JSON Data
https://dummyjson.com/todos

# Технологии

- iOS: 16.6+
- Swift: 5.9
- Архитектура: VIPER
- Data Layer: Core Data
- UIKit (без сторонних библиотек)
- Многопоточность: Обработка создания, загрузки, редактирования, удаления и поиска задач выполняется в фоновом потоке с использованием GCD. Интерфейс не блокируется при выполнении операций
- Unit тесты

# Возможности

- При первом запуске приложение загружает список задач из DummyJSON API
- Задача содержит: название, описание, дату создания и статус (выполнена/не выполнена)
- Изменение статуса задачи через чек-бокс (выполнена/не выполнена)
- Хранение данных в Core Data
- Добавление, удаление и редактирование задач  
- Поиск по названию задачи
